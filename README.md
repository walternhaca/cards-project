# CardsProject

Aplicação Angular para estudo e prática de conceitos como Class Binding, @Input, @Output, Event Binding, Property Binding, Componentização e Modularização.

## Tecnologias
O projecto foi desenvolvido em:

- [Angular16](https://angular.io/)
- [node18](https://nodejs.org/)

## Setup

Clone o projeto `git clone <url> `. Instale NPM packages `npm install`. Rode `npm run start`. Acesse no navegador: `http://localhost:4200/`

## Criando um novo componente

`npm run ng g c components/nome-do-componente`

## Padrão de Commits

Este projeto segue a convenção [Conventional Commits](https://www.conventionalcommits.org/pt-br/v1.0.0-beta.4/)
